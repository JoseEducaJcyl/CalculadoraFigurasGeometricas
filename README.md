# Calculadora de Figuras Geométricas

Este proyecto es una aplicación de consola en Java que permite calcular el área y el perímetro de varias figuras geométricas: **círculo**, **rectángulo** y **triángulo**.  
El programa utiliza Programación Orientada a Objetos (POO), herencia y manejo básico de excepciones.

## Funcionalidades
- Crear círculo, rectángulo o triángulo introduciendo sus datos.
- Calcular área y perímetro de cada figura.
- Guardar las figuras creadas en un array.
- Mostrar un resumen de todas las figuras creadas.
- Dibujar las figuras usando un menú adicional.
